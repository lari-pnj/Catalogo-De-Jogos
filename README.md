# Catalogo-De-Jogos
Este projeto é um catálogo de jogos desenvolvido durante a Imersão da Alura, utilizando HTML, CSS e JavaScript.

![imagem](/img/Captura%20de%20Tela%20(102).png)


O objetivo foi praticar conceitos fundamentais de desenvolvimento web, como:

Estruturação de páginas com HTML

Estilização e responsividade com CSS

Manipulação de dados e interação com o usuário usando JavaScript


\ Funcionalidades /

Busca de jogos pelo nome

Lista dinâmica de jogos

Layout responsivo

Interface simples e intuitiva


\ Tecnologias utilizadas /

HTML5

CSS3

JavaScript

\ Objetivo do projeto

Aprender e aplicar conceitos de front-end, reforçando lógica de programação e habilidades de construção de interfaces.
